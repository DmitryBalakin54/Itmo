      Warhammer 40,000: Rogue Trader - Founders Pack
      Warhammer 40,000: Rogue Trader - Limited Outfit
      Warhammer 40,000: Rogue Trader - Season Pass
      Warhammer 40,000: Rogue Trader - Voidfarer Pack
      Warhammer 40,000: Rogue Trader - Deluxe Pack
      Warhammer 40,000: Rogue Trader - Void Shadows
      3.В параметрах папок Windows убираешь галочку с пункта "Скрывать расширения для зарегистрированных типов файлов"
      4.Просто переименовываешь файл steam.info убирая в конце .txt и сохраняешь соглашаясь на изменение формата.
      5.Повторяешь этот пункт из комментария выше и готово:
      Затем создать файл по пути WH40KRT_Data\Plugins\x86_64\DLC.txt:
      2682660=Warhammer 40,000: Rogue Trader - Deluxe Pack
      2682670=Warhammer 40,000: Rogue Trader - Voidfarer Pack
