       Показать / Скрыть текст

      Сложность игры настраивается в файле "ServerSettings.ini", который находится в папке "Config".
      Сложность можно менять на лету, редактируя этот файл, либо зайдя в "настройки сервера" в "главном меню".
      Всё это проделывается только со знанием дела, либо опытным путём.
      Чтобы облегчить задачу, сделаем следующее:
      0. Сделать резервную копию папки "Saved" по пути "Conan Exiles\ConanSandbox\Saved".
      1. Зайти в папку "Saved" по пути "Conan Exiles\ConanSandbox\Saved".
      2. Скопировать и сохранить в другом месте эти файлы:
      -- все файлы с расширением .db(Data Base File) - это сохранения.
      3. После создания резервной копии сохранений, удалить из папки "Saved" следующее:
      -- все файлы с расширением .db(Data Base File) - это сохранения.
      -- папку "Logs".
      -- папку "Config".
      4. Зайти в главное меню игры(оно будет на английском)-->>зайти в "SETTINGS"-->>перейти на вкладку "GAMEPLAY"-->>в разделе
